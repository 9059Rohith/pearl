# Publication Interview Assessment

Synthetic assessment repo for evaluating engineering candidates.

## Scope

- `apps/api` — NestJS backend (pages, leads, brands, email)
- `apps/admin` — Next.js admin portal
- `apps/site` — Next.js public site
- `packages/shared` — Shared types

## Working Rules

- This is an assessment codebase — intentionally contains architectural issues
- All modifications are permitted during assessment development
